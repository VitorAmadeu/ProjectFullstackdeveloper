Sobre mim

Olá, meu nome é Vitor Hugo Amadeu e sou estudante de Engenharia da Computação na UTFPR. Atualmente, estou focado em aprimorar minhas habilidades no desenvolvimento web, com um grande interesse nas áreas de dados e backend, onde tenho maior afinidade.

Desde cedo, sempre fui fascinado pela tecnologia e pelo potencial que ela tem para transformar o mundo. Durante minha jornada acadêmica, descobri uma paixão especial pela programação e pelo desenvolvimento de sistemas robustos e eficientes. Isso me levou a explorar profundamente o desenvolvimento backend, onde encontro desafios estimulantes e oportunidades de aprendizado constante.

sempre em busca de novos conhecimentos e experiências que me ajudem a crescer como profissional e como pessoa. Meu objetivo é contribuir para projetos inovadores e impactantes, aplicando as melhores práticas e tecnologias disponíveis.

Quando não estou codificando, gosto de explorar o mundo dos dados, tentando entender como eles podem ser utilizados para resolver problemas reais e criar valor. Acredito que a combinação de uma base sólida em engenharia e um foco na prática constante é a chave para se tornar um desenvolvedor de sucesso.

Serviços Oferecidos

  1. Desenvolvimento de APIs (Application Programming Interfaces)

    • Criação de APIs RESTful: Desenvolvimento de APIs que seguem o estilo arquitetural REST, permitindo a comunicação entre diferentes partes de um sistema.
    • Desenvolvimento de APIs GraphQL: Implementação de APIs que utilizam GraphQL, oferecendo mais flexibilidade nas consultas e na manipulação de dados.

  2. Gerenciamento de Banco de Dados

    • Design de Banco de Dados: Estruturação e modelagem de bancos de dados relacionais (como MySQL, PostgreSQL) ou não relacionais (como MongoDB, Cassandra).
    • Otimização de Consultas: Ajuste e otimização de consultas para melhorar o desempenho do banco de dados.
    • Migração de Dados:Transferência e transformação de dados entre diferentes sistemas ou versões de bancos de dados.

  3. Desenvolvimento de Aplicações Servidor

    • Desenvolvimento de Aplicações Web: Criação de lógicas de negócios e funcionalidades complexas no servidor, utilizando linguagens como Node.js, Python (Django, Flask), Ruby (Rails), Java (Spring), entre outras.
    • Integração de Serviços de Terceiros: Conexão com APIs e serviços externos, como gateways de pagamento, serviços de envio de e-mail, redes sociais, etc.
